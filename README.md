# Cocktails Finder

Cocktails Finder showcases MVVM architecture. Applications fetches data form REST API service and displays it for user. There is several REST calls depending on user interaction.

Application consists of :

- MVVM
- Retrofit 2
- Dagger 2
- ViewModel
- View Binding
- Recycle View
